Contribuciones

Nos encanta que la comunidad nos ayude a mejorar y crear proyectos Code Club. Nos encantaría oír vuestros comentarios sobre las lecciones o recibir ayuda con la traducción de las lecciones a otros idiomas.

Comentarios sobre los proyectos

¿Por qué nos hace falta vuestra ayuda?

Existen numerosos modos con los que podéis contribuir al Code Club:

  • Ayudándonos a mejorar los proyectos existentes proporcionando vuestros comentarios
  • Sugiriendo correcciones de los proyectos
  • Escribiendo nuevos proyectos, ¡nunca hay demasiados!
  • Traduciendo los proyectos existentes para ayudarnos a expandirnos internacionalmente

¿Queréis contribuir?

¡Genial! Gestionamos todos nuestros cursos y proyectos desde GitHub. Si estáis listos y podéis ayudar, dirigiros a la página de GitHub del Code Club y consultar los documentos guía sobre las contribuciones.
