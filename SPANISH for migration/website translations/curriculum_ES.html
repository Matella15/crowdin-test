Currículo

Los proyectos de Code Club son guías paso a paso para que los niños creen animaciones, juegos, sitios web y mucho más. Los niños desarrollarán sus habilidades de programación según progresen por los proyectos y los retos contenidos son la oportunidad ideal para demostrar y aplicar lo aprendido.

  • Scratch
  • HTML & CSS
  • Python

Scratch

Scratch es un lenguaje de programación gráfico desarrollado por el grupo Lifelong Kindergarten y el Instituto de Tecnología de Massachusetts. Los niños pueden arrastrar y combinar bloques de códigos para crear una amplia variedad de programas, incluyendo animaciones, historias, instrumentos musicales y juegos. ¡En programación, es el equivalente de Lego!

HTML & CSS

HyperText Markup Language (HTML) es un lenguaje para describir y estructurar páginas web. Los documentos HTML pueden incluir, por ejemplo, texto, imágenes, audio y vídeos usando ´etiquetas´para organizar el contenido.
